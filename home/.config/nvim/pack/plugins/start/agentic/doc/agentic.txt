*agentic.txt*    Send file/selection refs to a claude pane in tmux

==============================================================================
INTRODUCTION                                              *agentic-introduction*

agentic forwards a reference to the current file (or visual selection) into a
running claude on a tmux pane, then focuses that pane. It talks to the tmux
server directly, so nvim need not be inside tmux itself.

Reference format mirrors GitHub-style anchors:

    @path/to/file              normal mode
    @path/to/file#L4-6         visual selection (range)
    @path/to/file#L4           visual selection (single line)

Paths are relative to the git root when available, else to cwd.

==============================================================================
TARGETING                                                     *agentic-targeting*

By default a send assumes nvim is inside tmux: it reads the current session
and finds claude panes within it. One match → sent. Several → you're prompted.
None (or nvim isn't in tmux) → a warning pointing you at :AgenticRegister.

:AgenticRegister overrides the session: it lists ALL sessions on the server
(works even when nvim isn't in tmux) so you can lock onto one; future sends
then look for claude in that session. Sticky until you re-register or the
session dies, at which point sends fall back to the current session.

Delivery is load-buffer + paste-buffer (no trailing Enter — review, then
submit yourself), followed by select-pane.

==============================================================================
COMMANDS                                                      *agentic-commands*

:AgenticSendFile                                            *:AgenticSendFile*
    Send the current buffer's file reference.

:AgenticSendSelection                                  *:AgenticSendSelection*
    Send the current visual selection as a line-range reference.

:AgenticRegister                                            *:AgenticRegister*
    Register against any tmux claude, overriding repo correlation.
